# Volume-Guardian

My first Chrome extension, created as a learning project.

## Overview

Volume Guardian automatically limits the volume of YouTube videos, helping prevent the volume from being increased beyond a predefined threshold.

This project was inspired by a real situation observed in an educational setting, where a child repeatedly increased the volume to the maximum level.

## Features

- ✅ Automatically monitors YouTube video volume.
- ✅ Restores the volume if it exceeds the configured limit.
- ✅ Lightweight and runs automatically in the background.

## Technologies

- JavaScript
- HTML
- Chrome Extensions (Manifest V3)

## Future Improvements

- Adjustable maximum volume (slider).
- Support for additional websites.
- Improved user interface.
- Administrator password to change settings.
- Chrome Web Store publication.

## Purpose

This project was created to practice JavaScript and learn how Chrome extensions work through a real-world accessibility use case.
